# Agent App

React Native / Expo mobile application for delivery agents.

## Features

- Agent authentication (Firebase Auth)
- Zone selection
- Barcode scanning and master data fetch
- Delivery record saving (Firestore)
- Offline sync support
- Background GPS tracking

## Getting Started

```bash
npm install
npx expo start
```

## Build (EAS)

```bash
eas build --platform android
eas build --platform ios
```
